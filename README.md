# Portfolio
##  Présentation

Ce projet consiste à concevoir et développer un **portfolio web personnel** en utilisant **HTML5 et CSS3 uniquement**.  
Il a été réalisé dans le cadre d’un **mini-projet de développement web** dont l’objectif est de mettre en pratique les notions fondamentales du développement front-end.

Le portfolio permet de présenter :

- mon profil académique
- mes compétences techniques
- quelques projets réalisés
- mon parcours universitaire
- des liens vers mes profils professionnels

Ce site constitue également une **première vitrine professionnelle** me permettant de présenter mon travail et mes compétences dans le domaine du développement et de l’informatique.

---

#  Objectifs du projet

Les objectifs principaux de ce mini-projet sont :

- Apprendre à structurer une page web avec **HTML5 sémantique**
- Créer une interface visuelle avec **CSS3**
- Utiliser **Flexbox et CSS Grid** pour la mise en page
- Concevoir un **design moderne et cohérent**
- Mettre en place un **site responsive adapté aux mobiles**
- Organiser un projet web proprement
- Publier le projet sur **GitHub**

---

#  Technologies utilisées

Les technologies utilisées dans ce projet sont :

- **HTML5** → structure du site
- **CSS3** → mise en forme et design
- **Flexbox** → alignement et organisation des éléments
- **CSS Grid** → structure des sections
- **Google Fonts** → typographie du site
- **Font Awesome** → icônes

Ces technologies permettent de créer un site web moderne sans utiliser JavaScript.
